# ホーム設定（Customizer）使い方メモ

対象テーマ: `washoen3`

## 設定場所
- 外観 > カスタマイズ > ホーム設定

## 設定できる画像と推奨サイズ
- ヒーロー画像: 推奨 1920×1080（16:9）
- 店舗カード画像（福中店）: 推奨 1200×800
- 店舗カード画像（塩町店）: 推奨 1200×800
- アクセス背景（任意）: 推奨 1920×1080
- お品書きアイコン画像（福中店・塩町店）: 推奨 48×48（正方形）。未設定時は既存のアイコンを表示。

各項目はメディアライブラリから画像を選択/アップロードしてください（添付IDで保存）。

## 反映箇所
- ヒーロー: トップのビジュアル背景として自動反映（`index.php`）
- 二店舗のご案内: 各カードのメイン画像に反映（未設定時はプレースホルダー）
- アクセス: セクションの背景として反映（未設定時は既存の背景グラデーション）
- お品書き: 各店舗カードのアイコンに反映（未設定時はFont Awesomeアイコン）

## 備考
- 画像は `home-hero`（1920x1080）/`home-card`（1200x800）サイズで出力、`srcset` 付きで自動レスポンシブ化されます。
- 未設定時は既存レイアウトのまま（デフォルト背景・プレースホルダー）表示され、崩れはありません。
- 変更が反映されない場合は、ブラウザ/サイトキャッシュのクリアをお試しください。
